# ShopZone 🛍️

A modern e-commerce web application built with **React, React Router, and Context API**. ShopZone provides a seamless shopping experience with dynamic product browsing, cart management, and smooth navigation without page reloads.

## 🚀 Features

* 🏠 Modern Landing Page
* 🛒 Add to Cart Functionality
* ❌ Remove Items from Cart
* 🔄 Global State Management using Context API
* 🧭 Client-Side Routing with React Router
* ⚡ Fast and Responsive User Interface
* 📱 Mobile-Friendly Design
* 🔍 Product Details Page
* 🛍️ Persistent Cart Across Routes

## 🛠️ Tech Stack

* React.js
* React Router DOM
* Context API
* JavaScript (ES6+)
* CSS3
* Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd shopzone
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:5173
```
